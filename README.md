# 🔐 Ciphers – Cryptography Algorithms in C++

This repository contains implementations of various **classical cipher algorithms** written in **C++**.
The goal of this project is to understand the working principles of basic cryptography techniques used for **encryption and decryption of messages**.

---

## 📌 About the Project

Cryptography is used to secure communication by converting readable information (plaintext) into an unreadable format (ciphertext).

This repository demonstrates how some of the **foundational cipher techniques** work internally.

It is useful for:

* Students learning **cryptography basics**
* Practicing **C++ programming**
* Understanding **encryption and decryption algorithms**

---

## 🚀 Implemented Ciphers

The repository currently includes implementations of:

* Caesar Cipher
* Vigenère Cipher
* Playfair Cipher
* Hill Cipher
* Rail Fence Cipher
* Substitution Cipher

Each cipher program demonstrates:

* Encryption process
* Decryption process
* User input handling

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/fhdshaikh33/CIPHERS.git
```

2. Navigate to the project folder

```
cd CIPHERS
```

3. Compile the program

```
g++ filename.cpp -o cipher
```

4. Run the program

```
./cipher
```
